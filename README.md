# 🖥️ Computer Architecture Mastery
> 본 리포지토리는 도서 [**컴퓨터 구조 및 설계 6판 (ARM 편)**]를 기반으로, 리눅스 시스템 엔지니어, 디바이스 드라이버, 그리고 임베디드 소프트웨어 개발자 관점에서 필수적인 하드웨어 지식을 현업과 연계하여 정리한 기록입니다.단순한 하드웨어 이론을 넘어, 이 지식이 실제 운영체제와 컴파일러 성능 최적화에 어떻게 반영되는지 실무적인 트러블슈팅 및 분석 과정과 함께 담고 있습니다.

## 💻 개발 및 분석 환경
하드웨어 아키텍처의 이해도를 높이기 위해 제한된 환경 및 실제 ARM 타겟에서 실습과 검증을 병행합니다.
* Reference: 컴퓨터 구조 및 설계 6판 (arm 편)
* Target Architecture: `ARM64`
* Analysis Host: `raspberry4`
* Kernel Version: `6.18.35-v8+`
